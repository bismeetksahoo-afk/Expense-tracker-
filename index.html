How to Set Up Firebase
1
Go to firebase.google.com
Sign in with your Google account → click "Get Started"
2
Create a project
Click "Add project" → name it "xpensetrack" → skip Google Analytics → Create
3
Add a Web App
Click the </> icon → name it "xpensetrack-web" → Register app → copy the firebaseConfig object
4
Set up Firestore
In left sidebar → Build → Firestore Database → Create database → Start in test mode → Choose any region → Enable
5
Paste config above
Go to Setup tab in this app → paste the config JSON → Save & Connect
6
Share the link
Share your GitHub Pages URL with family. Everyone uses the same link — data syncs instantly!
